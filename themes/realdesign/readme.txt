== Description ==
This theme is a very simplistic design that can be used as a blog or for business and includes a Contact Us form. Users have the option to use the colors they choose for the background, menu bar, and sidebar. When using as a blog, Posts have share buttons for Facebook, Twitter, and Google+. 

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in RealDesign in the search form and press the 'Enter' key in your keyboard.
3. Click on the 'Activate' button to use your new theme right away.
4. Navigate to Appearance > Customize in your admin panel.

== Notes ==
* This theme includes a sidebar on the right side that contains Wordpress Widgets. 
  The the sidebar can be enabled or disabled using the theme customizer. When enabled,
  the sidebar is only available in the single.php and index.php files.
* This theme currently does not display the category for posts or pages.
